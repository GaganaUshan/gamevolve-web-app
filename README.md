
  ## Gaming News Web App

  This is a code bundle for Gaming News Web App. The original project is available at https://www.figma.com/design/Lk6JkhRLPW8JjHfwwPO0Mc/Gaming-News-Web-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  A simple **frontend-only News Management Application** built using **React, Vite, TypeScript, and Tailwind CSS**.  
The app allows users to **add, edit, and delete news articles** with data saved in **browser localStorage** (no backend).
